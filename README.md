# Personal Introduction Program

## Project Overview
This is a beginner Python program that asks the user for personal information and displays a welcome message.

## Features
- Uses input() to collect user data
- Stores information using variables
- Displays formatted output using print()

## Setup Instructions
1. Install Python
2. Open terminal
3. Run the program using:
   python personal_intro.py

## What I Learned
- Python basics
- Variables and user input
- Simple program structure
